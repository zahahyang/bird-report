🐦 鳥類通報系統 — Bird Fortune Report System
楊媽媽通靈研究所 x 宇宙情報局 聯合開發

「科學的盡頭是玄學。」—— 楊媽媽

一套基於傳統風水學與民間鳥類預兆研究的即時運勢追蹤系統。透過記錄日常鳥類目擊事件，結合古籍文獻中的吉凶判讀，為使用者提供每日運勢分析。


📋 專案概述
在全球經濟動盪、求職市場嚴峻的 2026 年，傳統的理性分析已不足以應對不確定性。本系統回歸東方玄學智慧，以「鳥類出沒」作為運勢指標，建立一套可量化、可追蹤、可驗證的預測框架。
核心理念
鳥落旺地，財聚旺鄉 —— 鳥類對磁場與氣場極為敏感，其出沒行為可作為環境能量的自然指標
科學驗證精神 —— 內建 KPI 考核系統，每日預測皆可回溯驗證，計算準確率
在地化研究 —— 以新北市三重區為田野調查基地，收錄 20 種在地常見鳥種


✨ 功能特色
即時通報
下拉選單快速選擇鳥種，支援備註欄位記錄目擊情境
通報後即時顯示該鳥種的風水等級、古籍典故與「楊媽媽判讀」結果
8 種常見鳥類附有手繪 SVG 插圖，通報時帶飛入動畫
鳥種風水圖鑑
收錄 20 種鳥類，依風水等級分為 SSS / SS / S / A / B / C / D 七級
每種鳥附有風水意義說明，參考古籍文獻與民間傳說
記錄各鳥種累計目擊次數
歷史紀錄 & KPI 考核
所有通報紀錄以日期分組，永久保存於本地端
每日可回溯驗證「楊媽媽的預測準不準」（準了 ✔ / 沒準 ✘）
自動計算系統準確率，以數據驗證玄學
設計規格
Pantone 2026 年度色 Cloud Dancer 配色方案
完整 RWD 響應式設計，手機優先
支援深色模式
可加入手機主畫面作為獨立 Web App
純前端架構，資料以 localStorage 存儲，無需後端

🔬 研究方法論
文獻基礎
《本草綱目》李時珍：「喜鵲靈能報喜，故謂之喜」
民間俗語：「鳥落旺地，財聚旺鄉」「燕子不進愁家門」
風水學：鳥類對氣場敏感，選擇築巢地點反映環境能量
斑鳩歷史地位：古代皇帝欽封之福星鳥
資料收集
田野調查地點：新北市三重區
觀測設備：Samsung S25 Ultra（長焦鏡頭）
首席分析師：楊媽媽（資歷數十年，準確率持續驗證中）


🛠️ 技術架構
前端：純 HTML / CSS / JavaScript，無框架依賴
後端：Firebase Firestore（雲端資料庫）+ Firebase Authentication（Google 登入）
儲存：Firebase Firestore（主要）+ localStorage（離線備援）
設計系統：Pantone 2026 Cloud Dancer 色系 + Noto Sans TC + Zen Maru Gothic
插圖：手繪 SVG 鳥類插圖（20 種全收錄）
部署：GitHub Pages
架構：前後台分離（index.html 前台 + admin.html 後台）


📱 使用方式
前台（通報）
前往 zahahyang.github.io/bird-report
手機瀏覽器開啟 → 「分享」→「加入主畫面」
看到鳥 → 打開 App → 選鳥種 → 選地點 → 按「通報！」
隔天回來驗證楊媽媽準不準
後台（Dashboard）
前往 zahahyang.github.io/bird-report/admin.html
使用授權 Google 帳號登入
查看 Dashboard 數據、鳥種圖鑑、玄學資料庫
